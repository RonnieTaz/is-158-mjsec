## Run Locally

Clone the project

```bash
  git clone https://github.com/RonnieTaz/is-158-mjsec.git
```

Create database with name `mjsec`

Install dependencies

```bash
  composer install
```

Run Migrations

```
    php vendor/bin/phoenix migrate
```
Start the server

```bash
  php -S localhost:8080 -t public/
```
